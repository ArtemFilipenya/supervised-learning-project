Прогноз снижения покупательской активности клиентов
Краткое описание
Цель проекта — предсказать снижение покупательской активности клиентов интернет-магазина «В один клик» и на основе модели выполнить сегментацию клиентов для проактивных мер удержания.

Основные задачи:
* Сформулировать и решить задачу бинарной классификации: целевой класс — “Снизилась активность”.
* Построить воспроизводимый пайплайн предобработки и обучения моделей.
* Сравнить несколько моделей и выбрать лучшую по бизнес-релевантным метрикам.
* Проанализировать важность признаков и дать рекомендации по удержанию.
* Провести сегментацию клиентов и выделить критичный сегмент “Ценные, в зоне риска”.
Данные

Источники: поведенческие логи и транзакционные данные, агрегированные в таблицы:
market_file, market_money, market_time.
Ключевые шаги с данными:
* Отбор клиентов с активностью ? 3 месяцев.
* Объединение таблиц и формирование признаков (в т.ч. по периодам).
* Разделение на численные и категориальные признаки.

Лучший результат:
* Модель: SVC(C=1.0) в пайплайне со StandardScaler и OneHotEncoder.
* Результаты на тестовой выборке:
* F1 (класс “Снизилась”): 0.79
* Recall (класс “Снизилась”): 0.69
* Precision (класс “Снизилась”): 0.93
* ROC-AUC: 0.87
